## 🚀 Installation

<details open>
<summary><b>For Debian-based Systems (Kali/Ubuntu)</b></summary>

```bash
# Clone the repository
git clone https://github.com/jcrvnx/orhenx.git

# Set permissions
chmod -R 755 orhenx

# Navigate to directory
cd orhenx

# Run installer
sudo bash install.sh

# Launch ORHENX
sudo orhenx
```
</details>
